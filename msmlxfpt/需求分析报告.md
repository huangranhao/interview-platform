## 一、引言

本报告详细描述面试模拟训练平台的功能需求和非功能需求，作为项目开发的核心依据。

## 四、数据需求

### 4.1 数据实体

| 实体 | 描述 |
| :--- | :--- |
| 用户 | 用户基本信息、注册时间、角色 |
| 面试记录 | 面试ID、用户ID、岗位类型、难度、时间 |
| 评估报告 | 面试ID、各项评分、建议内容 |
| 题目 | 题目ID、岗位类型、难度、问题、答案 |

### 4.2 数据流

```mermaid
flowchart LR
    A[用户] -->|发起面试| B[面试模块]
    B -->|生成问题| C[题库]
    B -->|录制视频| D[视频存储]
    B -->|提交回答| E[AI评估]
    E -->|生成报告| F[评估报告]
    F -->|存储| G[数据库]
```
